# 調査メモ: JANコード検索 API

## 目的
JANコードをキーとして外部商品検索 API から商品情報を取得し、API 応答に正規化して返す。

## 主要な技術判断
- 検索クライアントは `HttpClient` を使用し、`HomeFinder.Infrastructure` に実装する。
- アプリケーション層の公開インターフェイスは `DotNext.Result<T>` を返す。
- 複数ヒット時は外部 API のデフォルト順序で先頭要素を採用する。

## 想定レスポンス差異への対応
外部 API のフィールド名揺れを吸収するため、以下の候補を優先順で解釈する。
- 名前: `name` / `itemName` / `title`
- メーカー: `manufacturer` / `manufacturerName` / `makerName` / `brand`
- 価格: `price` / `itemPrice` / `minPrice`

## エラーマッピング
- 不正JAN: `400`
- 商品なし: `404`
- レートリミット: `429`（透過）
- 認証失敗: `500`
- タイムアウト: `503`
- その他外部 API 失敗: `500`

## 非機能補足
- 本イテレーションでは性能試験の実行実装は対象外（設計資料のみ）。
