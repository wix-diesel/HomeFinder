# Contract: 設定画面表示契約

本契約は、設定画面の表示要件とスコープ境界を定義する。

## 1. 表示構成

- 画面は `design/settings.html` の情報構成に準拠する。
- 最低限含む要素:
  - 画面タイトル
  - セクション見出し
  - 設定項目ラベル/補足説明
  - 戻る導線または一覧へ戻る導線

## 2. 文言要件

- 可視テキストは日本語で表示する。
- 英語の固定文言は表示しない。
- 見出し、説明、ボタン、補助文言を日本語化対象に含める。

## 3. 項目挙動要件

- 設定項目行は本機能では表示のみ。
- 各項目の遷移先実装は行わない。
- 表示のみ項目は `actionType=display_only` として扱う。

## 4. デザイン整合

- 一覧画面 (`design/items_list.html`) と視覚的一貫性を持つ。
- ヘッダー、余白、情報ブロック配置は参照デザインとの差異を最小化する。

## 5. テスト観点

- 画面内の可視文言日本語率 100%。
- 設定項目操作で外部遷移が発生しないこと。
- 一覧から遷移後に設定画面が安定表示されること。
