# Contract: 設定導線 UI

本契約は、一覧画面から設定画面へ遷移する導線要素の UI/操作仕様を定義する。

## 1. 対象画面

- 一覧画面: `/items`
- 設定画面: `/settings`

## 2. NavigationButton 契約

- 目的: 一覧画面右上から設定画面へ 1 操作で到達させる。
- 要素要件:
  - 位置: 一覧画面ヘッダー領域の右上
  - アイコン: 歯車 (`settings`) を表す視覚表現
  - ラベル: 支援技術向けの日本語ラベル (例: 「設定画面へ移動」)
  - フォーカス: キーボード到達可能で、可視フォーカススタイルを持つ
- 操作要件:
  - `click` で `/settings` へ遷移
  - `Enter` または `Space` 実行で `/settings` へ遷移

## 3. ルート遷移契約

- ルーター名: `settings`
- Path: `/settings`
- 失敗時契約:
  - ルート解決失敗時は一覧画面に留まり、再試行可能な状態を維持する

## 4. アクセシビリティ要件

- スクリーンリーダーで導線目的を識別できる名前を提供する。
- フォーカスインジケータは背景と十分に識別できる。
- 導線要素は装飾専用要素ではなく、操作可能要素として実装する。

## 5. 非対象

- 設定画面内項目から他ページへの遷移実装。
- 設定保存 API の追加。
