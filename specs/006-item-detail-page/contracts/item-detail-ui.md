# Contract: Item Detail UI

## 画面要件

- デザイン基準: design/items_detail.html
- 表示言語: 日本語
- スコープ外:
  - 直近の変更履歴表示は行わない
  - 右下履歴ボタンは非活性（クリック不可）

## 画面要素

### 1. 詳細情報セクション
- 一覧で選択された単一アイテムの情報を表示する

### 2. 右上 3点リーダー
- クリックでアクションメニューを表示
- メニュー項目:
  - 編集
  - 削除

### 3. 左下 編集ボタン
- クリックで編集ページへ遷移

### 4. 削除確認ダイアログ
- 削除押下時に表示
- 操作:
  - 削除確定
  - キャンセル

### 5. 右下 履歴ボタン
- 常時非活性
- ポインターイベントを受け付けない

## Interaction Rules

- 一覧 -> 詳細: アイテム選択で遷移
- 編集導線:
  - 3点リーダー内「編集」
  - 左下「編集」
  - どちらも同一編集ページへ遷移
- 削除導線:
  - 3点リーダー内「削除」 -> 確認ダイアログ
  - 確定時のみ削除 API 実行
  - 成功後は一覧へ遷移
- 競合/対象消失:
  - 失敗メッセージ表示後に一覧へ遷移

## Reuse Policy

- 既存画面に同等コンポーネントがある場合は再利用を優先する。
- 新規実装は既存コンポーネント設計との整合を維持する。
